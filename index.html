Regras

Cada pequeno tabuleiro de jogo da velha 3 × 3 é definido como um subtabuleiro, e o maior tabuleiro 3 × 3 é definido como o tabuleiro principal.

O jogo começa com o X jogando onde quiser em qualquer um dos 81 espaços vazios. Esta jogada “envia” seu oponente para a sua localização relativa. Por exemplo, se o X jogou no quadrado superior direito do seu subtabuleiro, então o O precisa jogar a seguir no subtabuleiro no canto superior direito do tabuleiro principal. O O pode então jogar em qualquer um dos nove espaços disponíveis nesse subtabuleiro, cada jogada enviando o X para um subtabuleiro diferente.

Se uma jogada é feita de forma a ganhar um subtabuleiro pelas regras do jogo da velha normal, então todo o subtabuleiro é marcado como uma vitória para o jogador no tabuleiro principal.

Uma vez que um subtabuleiro é ganho por um jogador ou está completamente preenchido, não se pode mais jogar nesse tabuleiro. Se um jogador é enviado para tal tabuleiro, então esse jogador pode jogar em qualquer outro tabuleiro.

O jogo termina quando um jogador ganha o tabuleiro principal ou não há mais jogadas legais restantes, caso em que o jogo é um empate.
